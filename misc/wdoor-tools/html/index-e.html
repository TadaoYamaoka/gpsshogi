How to participate

CSA client

  • If your client speaks the standard CSA protocol over TCP/IP,
    • connect to wdoor.c.u-tokyo.ac.jp:4081
    • login with special password "floodgate-300-10F,trip", where "trip" should be replaced by your own secret string
    • wait for game start (the server tries to pair players every half an hour)
  • If your client speaks the extended CSA protocol over TCP/IP, just specify "floodgate-300-10F" for gamename.
  • If your client can only communicate with human, you can use a bridge:
    • Shogi Framework Implements CSA Protocol (the author: J. Takada)

USI client

  • If your client speaks usi protocol, you can use a bridge program to connect the Floodgate.
    • The Shogi-server comes with a sample script 'usiToCsa.rb' written in Ruby, which has yet to support the Fischer time management. (document)
    • csa2usi.pl (the author: the developer of GPS Shogi)
    • ShogiDokoro (the author: anonymous)
